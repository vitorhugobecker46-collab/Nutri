NutriSnap — Protótipo estático
Conteúdo:
- index.html
- styles.css
- script.js
- assets/logo.svg

Como publicar no Vercel (rápido):
1) Extraia esta pasta.
2) Crie conta em https://vercel.com (use seu e-mail).
3) No painel, New Project -> Importar Repositório OU Deploy -> Upload (arraste a pasta extraída).
4) Vercel detecta site estático e faz deploy automaticamente.
5) Pronto: você terá um link tipo https://nutrisnap-yourname.vercel.app

Observações:
- Este é um protótipo estático (simula análise). Para análise real por IA, é preciso criar backend com modelos de visão (ex: TensorFlow/Detectron ou APIs comerciais) e um serviço de pagamento para o Premium.
